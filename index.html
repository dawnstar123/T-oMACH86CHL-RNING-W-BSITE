What is machine learning?
Machine learning is a type of artificial intelligence (AI) that provides computers with the ability to learn without being explicitly programmed. Machine learning focuses on the development of computer programs that can teach themselves to grow and change when exposed to new data.

Applications of supervised machine learning
Supervised machine learning is when an algorithm arranges data based on given data sets some examples of this are :
  • Application that can calculate house prices based on previous market prices
  • Application that takes medical data for diseases and diagnoses based on patterns it finds.
  • Game artificial intelligence that can adapt to user action based on previous history.

As you can see supervised machiene learning is primarily based on using previous data sets to output a value that corresponds to the input. This is typically done using using linear algebra or a

There are two types of supervisd machiene learning classification and regresssion both types are used extensiveliy in developing applications as they both had common uses.

  • In classification, the goal is to assign a class (or label) from a finite set of classes to an observation. That is, responses are categorical variables. Applications include spam filters, advertisement recommendation systems, and image and speech recognition. Predicting whether a patient will have a heart attack within a year is a classification problem, and the possible classes are true and false. Classification algorithms usually apply to nominal response values.
  • In regression, the goal is to predict a continuous measurement for an observation. That is, the responses variables are real numbers. Applications include forecasting stock prices, energy consumption, or disease incidence.

Supervised machiene learning algorithms can be primary used for prediction based tasks and applications.

Unsupervised Machiene learning

Unsupervised learning is where you you only have input data (X) and no corresponding output variables.

The goal for unsupervised learning is to model the underlying structure or distribution in the data in order to learn more about the data. These are called unsupervised learning because unlike supervised learning above there is no correct answers and there is no teacher. Algorithms are left to their own devises to discover and present the interesting structure in the data.

Unsupervised learning problems can be further grouped into clustering and association problems.

  • Clustering: A clustering problem is where you want to discover the inherent groupings in the data, such as grouping customers by purchasing behavior.
  • Association: An association rule learning problem is where you want to discover rules that describe large portions of your data, such as people that buy X also tend to buy Y.
